

# HUD デザイン改善プラン

現在のHUDはシンプルな半透明パネルのみ。SF/AR風のプロフェッショナルなデザインに進化させる。

## 変更内容

### 1. CSS (`src/index.css`) — 新しいアニメーションとスタイル追加

- **グラデーションボーダー**: `hud-panel` にシアン/ティール系のグロー付きボーダー
- **コーナーマーク**: AR風の四隅ブラケット装飾（疑似要素）
- **スキャンライン改善**: より洗練されたグラデーション + グロー効果
- **パルスドット**: ステータスインジケーター用の呼吸アニメーション
- **テキストグロー強化**: シアン系のネオングロー
- **新クラス追加**:
  - `.hud-corner-marks` — 四隅にL字ブラケット
  - `.hud-glow-border` — ティール色のグローボーダー
  - `.hud-status-dot` — 呼吸するステータスドット
  - `.scanning-line` の改善 — より太く、グロー付き

### 2. GuidePage (`src/pages/GuidePage.tsx`) — レイアウトとビジュアル強化

- **トップバー**: セーフエリア対応 (`pt-safe`)、左右に情報を分割表示（シーン名 + ステータスドット）
- **ビネット効果**: カメラフィード上に上下グラデーションオーバーレイ（文字の可読性向上）
- **指示パネル**: コーナーマーク付き、左にアクセントライン
- **ボトムコントロール**: ボタンに `rounded-2xl` + ホバーグロー + ラベルテキスト追加
- **スキャンライン**: 幅を太く、カラーをシアン系に
