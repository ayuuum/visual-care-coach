

## ケア対象者情報をガイド画面にハードコード表示

データベースは使わず、ダミーデータで対象者情報（名前・服薬・注意事項）をガイド画面のHUDに表示します。

### 実装内容

**1. `src/components/PatientInfoPanel.tsx` (新規)**
- タップで展開/折りたたみできるパネル
- 名前、年齢、介護度、服薬リスト（バッジ）、注意事項（⚠付き）を表示
- 半透明backdrop-blur、カメラ映像の上に重ねる
- ダミーデータをコンポーネント内にハードコード

```text
┌──────────────────────────────┐
│ [Scene]          [▾ 田中太郎]│ ← タップで展開
│                  │ 85歳 要介護3│
│                  │ 💊 ワーファリン│
│                  │ 💊 アムロジピン│
│                  │ ⚠ 転倒注意  │
│                  │ ⚠ 右膝に痛み│
│                  └───────────│
│        カメラ映像             │
│  ┌─────────────────────┐    │
│  │ AI指示               │    │
│  └─────────────────────┘    │
│    [🔊]  [⏹]               │
└──────────────────────────────┘
```

**2. `src/pages/GuidePage.tsx` (修正)**
- PatientInfoPanelをトップバー右側に追加
